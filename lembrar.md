# Lembrete

Pedir as credenciais do Supabase e adicioná-las ao arquivo `.env.local`.

As credenciais necessárias são:
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`

Você pode encontrá-las no painel do Supabase, em "Configurações" > "API".

Criar uma chave de API gratuita do TinyMCE para que o editor de notícias funcione. https://www.tiny.cloud/
