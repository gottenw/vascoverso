# 📝 Guia do Editor de Notícias - VascoVerso

## 🎯 Visão Geral

O novo editor de notícias foi completamente reformulado para ser **super intuitivo** e **fácil de usar**. Não é necessário conhecimento técnico!

## 🚀 Como Usar o Editor

### 1. **Criar Nova Notícia**

1. No painel admin, clique em "Nova Notícia"
2. Preencha o título
3. Faça upload da imagem principal (a que aparece no destaque)
4. Escreva o conteúdo usando o editor

### 2. **Barra de Ferramentas do Editor**

O editor possui uma barra de ferramentas visual com os seguintes botões:

#### 📋 Desfazer/Refazer
- **↶ Desfazer**: Volta a última ação
- **↷ Refazer**: Refaz uma ação desfeita

#### 📄 Títulos
- **H1, H2, H3**: Cria títulos de diferentes tamanhos
  - H1 = Título grande
  - H2 = Título médio
  - H3 = Título pequeno

#### ✍️ Formatação de Texto
- **B** (Negrito): Deixa o texto em negrito
- **I** (Itálico): Deixa o texto em itálico
- **U** (Sublinhado): Sublinha o texto
- **S** (Tachado): Risca o texto

#### 📐 Alinhamento
- **⬅** Alinhar à esquerda
- **⬌** Centralizar
- **➡** Alinhar à direita

#### 📋 Listas
- **•** Lista com marcadores (bullets)
- **1.** Lista numerada

#### 🖼️ Imagens e Links
- **🖼️ Botão Verde (Imagem)**: Clique para adicionar imagens no meio do texto
- **🔗 Link**: Selecione o texto e clique para transformar em link

### 3. **Upload de Imagem Principal**

1. Na seção "Imagem Principal da Notícia"
2. Clique na área tracejada
3. Selecione a imagem do seu computador
4. Aguarde o upload (aparecerá uma mensagem)
5. A imagem será exibida
6. Para remover, clique no **X vermelho** no canto

### 4. **Adicionar Imagens no Conteúdo**

1. Posicione o cursor onde quer a imagem
2. Clique no **botão verde com ícone de imagem** 🖼️
3. Selecione a imagem
4. A imagem será inserida automaticamente

### 5. **Adicionar Links**

1. Selecione o texto que deseja transformar em link
2. Clique no botão de **link** 🔗
3. Digite a URL (exemplo: https://www.vasco.com.br)
4. Pressione OK

### 6. **Salvar a Notícia**

1. Clique em **"Publicar Notícia"** (ou "Atualizar Notícia" se estiver editando)
2. Aguarde a confirmação
3. Pronto! A notícia está publicada

## 💡 Dicas Importantes

### ✅ Boas Práticas
- **Sempre adicione uma imagem principal** - ela aparece na página inicial
- **Use títulos (H1, H2, H3)** para organizar o conteúdo
- **Adicione imagens relevantes** no meio do texto
- **Use negrito** para destacar informações importantes
- **Faça listas** para tornar o conteúdo mais legível

### ⚠️ Cuidados
- Imagens muito grandes podem demorar para carregar
- Recomendamos imagens de até 2MB
- Use imagens em formato JPG, PNG ou WEBP
- Revise o texto antes de publicar

## 🆘 Problemas Comuns

### Imagem não carrega?
- Verifique se o arquivo não está muito grande (máx 5MB)
- Certifique-se de que é uma imagem (JPG, PNG, WEBP)

### Botão desabilitado?
- Aguarde até que o upload da imagem termine
- Verifique se preencheu o título (obrigatório)

### Editor não funciona?
- Recarregue a página
- Limpe o cache do navegador
- Entre em contato com o suporte técnico
